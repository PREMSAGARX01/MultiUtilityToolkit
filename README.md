# PR7 - Multi Utility Toolkit

## Overview

The Multi Utility Toolkit is a Python-based project developed to demonstrate the use of Python modules, packages, custom modules, and built-in libraries. The toolkit provides various utilities including date and time operations, mathematical calculations, random data generation, UUID generation, file handling, unit conversions, and dynamic module exploration.

This project demonstrates Python modular programming concepts and proper package organization.

## Features

### Date and Time Operations
- Display current date and time
- Calculate difference between two dates
- Format dates using strftime()
- Stopwatch functionality
- Countdown timer

### Mathematical Operations
- Calculate factorials
- Compound interest calculations
- Trigonometric calculations
- Area calculations for geometric shapes

### Random Data Generation
- Generate random numbers
- Generate random lists
- Create random passwords
- Generate OTPs

### UUID Generation
- Generate unique identifiers using UUID4

### File Operations
- Create files
- Write data to files
- Read file contents
- Append data to existing files

### Unit Conversion
- Kilometer to Mile conversion
- Celsius to Fahrenheit conversion

### Dynamic Module Exploration
- Explore module attributes using dir()
